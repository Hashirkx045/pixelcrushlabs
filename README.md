# PixelCrushLabs
PixelCrushLabs – A free, browser-based image compression and conversion tool. Compress, convert, and resize images instantly without uploading anything. Supports JPG, PNG, WEBP, AVIF, BMP, GIF, and more. Privacy-first, no tracking, and fully client-side.

## ✨ Features

- **Drag & drop** or click to upload images
- **Convert between formats**: JPEG, PNG, WEBP, AVIF, BMP, GIF
- **Adjust compression quality** (for lossy formats)
- **Resize images** with optional aspect ratio lock
- **Live preview** with original vs. processed side-by-side
- **Real-time file size** display
- **5-second animated download** with spinner
- **Fully responsive** – works on mobile, tablet, and desktop
- **No tracking, no uploads** – your images stay on your device

## 📦 Installation & Usage

1. Clone the repository or download the `index.html` file.
2. Open `index.html` in any modern web browser (Chrome, Firefox, Edge, Safari).
3. Drag an image onto the upload area or click to browse.
4. Adjust format, quality, and size as desired.
5. Click "Download processed image" – after a 5‑second animation, your file will be saved.

No build steps, no dependencies to install. Just a single HTML file.
